Bab 5: Arsitektur Cloud Computing

Struktur, komponen, dan alur kerja sistem cloud computing

1. Pengertian Arsitektur Cloud

Arsitektur cloud computing adalah desain struktur sistem cloud yang menunjukkan bagaimana komponen cloud saling terhubung dan bekerja untuk menyediakan layanan kepada pengguna.

2. Komponen Utama Arsitektur Cloud

  • Front-End Cloud
  • Back-End Cloud
  • Cloud Server
  • Cloud Storage
  • Cloud Network
  • Manajemen & Monitoring

3. Diagram Arsitektur Cloud Computing

Diagram berikut menunjukkan alur kerja dan hubungan antar komponen dalam arsitektur cloud computing.

User
↓
Front-End
(Browser / App)
↓
Internet / Cloud Network
↓
Cloud Server
Cloud Storage
Database
↓

Manajemen & Monitoring
Diagram ini menunjukkan bahwa pengguna tidak berinteraksi langsung dengan server, melainkan melalui front-end dan jaringan cloud.

4. Alur Kerja Arsitektur Cloud

  • Pengguna mengakses aplikasi melalui browser
  • Permintaan dikirim lewat internet
  • Cloud server memproses permintaan
  • Data diambil atau disimpan ke cloud storage
  • Hasil dikirim kembali ke pengguna

5. Manfaat Arsitektur Cloud

  • Skalabilitas tinggi
  • Efisiensi biaya
  • Keamanan terpusat
  • Kemudahan pengelolaan
