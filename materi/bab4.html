Bab 4: Virtualisasi dan Container

Fondasi utama teknologi cloud computing modern.

1. Pengertian Virtualisasi

Virtualisasi adalah teknologi yang memungkinkan satu perangkat keras fisik menjalankan beberapa sistem operasi dan aplikasi secara bersamaan menggunakan mesin virtual (Virtual Machine).

Dengan virtualisasi, sumber daya seperti CPU, RAM, dan storage dapat digunakan secara lebih efisien.

2. Cara Kerja Virtualisasi

Virtualisasi bekerja dengan bantuan perangkat lunak yang disebut Hypervisor. Hypervisor bertugas membagi dan mengelola sumber daya perangkat keras untuk setiap mesin virtual.

Alur Kerja:

  • Server fisik menyediakan sumber daya
  • Hypervisor mengelola hardware
  • Mesin virtual berjalan di atas hypervisor
  • Setiap VM memiliki OS dan aplikasi sendiri

3. Jenis Hypervisor

a. Hypervisor Tipe 1 (Bare Metal)

Hypervisor yang berjalan langsung di atas hardware tanpa sistem operasi tambahan.

  • Performa tinggi
  • Stabil dan aman
Contoh: VMware ESXi, Microsoft Hyper-V, Xen

b. Hypervisor Tipe 2 (Hosted)

Hypervisor yang berjalan di atas sistem operasi host.

  • Mudah digunakan
  • Cocok untuk pembelajaran
Contoh: VirtualBox, VMware Workstation

4. Virtual Machine (VM)

Virtual Machine adalah komputer virtual yang berjalan di dalam komputer fisik dan memiliki sistem operasi serta aplikasi sendiri.

Kelebihan VM:

  • Isolasi sistem tinggi
  • Mendukung berbagai OS
  • Cocok untuk aplikasi besar

Kekurangan VM:

  • Menggunakan resource besar
  • Waktu boot lebih lama

5. Container

Container adalah teknologi virtualisasi tingkat aplikasi yang memungkinkan aplikasi berjalan secara terisolasi tanpa perlu sistem operasi terpisah.

Container lebih ringan dan cepat dibandingkan Virtual Machine.

Karakteristik Container:

  • Ringan dan cepat
  • Menggunakan OS host
  • Mudah dipindahkan

6. Docker dan Kubernetes

Docker

Docker adalah platform untuk membuat, menjalankan, dan mengelola container.

Kubernetes

Kubernetes digunakan untuk mengatur dan mengelola banyak container dalam skala besar.

Docker + Kubernetes adalah kombinasi utama dalam cloud modern.

7. Perbandingan Virtual Machine dan Container

  • VM: Lebih aman, resource besar
  • Container: Ringan, cepat, efisien
Pemilihan VM atau container tergantung kebutuhan aplikasi dan skala sistem.
