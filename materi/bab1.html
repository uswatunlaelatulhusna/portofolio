Bab 1: Pengantar Cloud Computing

Memahami konsep dasar cloud computing sebagai fondasi pembelajaran.

1. Pengertian Cloud Computing

Cloud Computing adalah teknologi yang memungkinkan pengguna untuk mengakses dan menggunakan sumber daya komputasi seperti server, penyimpanan data (storage), database, jaringan, dan aplikasi melalui internet tanpa harus memiliki atau mengelola perangkat keras secara langsung.

Contoh sederhana cloud computing adalah Google Drive, Gmail, dan Google Docs yang dapat diakses kapan saja melalui internet.

2. Sejarah Singkat Cloud Computing

Konsep cloud computing berkembang dari teknologi komputasi terdistribusi dan virtualisasi. Seiring meningkatnya kebutuhan akan efisiensi dan skalabilitas, layanan cloud mulai digunakan secara luas oleh perusahaan dan individu.

3. Karakteristik Cloud Computing

  • On-demand self-service – pengguna dapat mengatur layanan sendiri
  • Broad network access – dapat diakses melalui berbagai perangkat
  • Resource pooling – sumber daya digunakan bersama
  • Rapid elasticity – kapasitas dapat ditambah atau dikurangi
  • Measured service – pembayaran sesuai penggunaan

4. Kelebihan dan Kekurangan Cloud Computing

Kelebihan

  • Tidak memerlukan investasi hardware besar
  • Akses fleksibel dari mana saja
  • Skalabilitas tinggi

Kekurangan

  • Ketergantungan pada koneksi internet
  • Isu keamanan dan privasi data
  • Kontrol sistem terbatas
