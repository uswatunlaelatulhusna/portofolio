Instruksi cepat:
1. Masukkan gambar Anda ke folder `img/` sesuai nama file di atas.
2. Buka `index.html` di browser.
3. Untuk mengubah teks (profil, pengalaman, kontak, skill), edit `index.html`.
4. Untuk mengubah gaya, edit `css/styles.css`.
5. Interaksi kotak materi diatur di `js/script.js`. Kotak memiliki `href` (contoh: `#material1`) untuk akses langsung.
